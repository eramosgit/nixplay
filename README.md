# 1. Kth Largest Element Finder

Node.js + TypeScript implementation to find the **kth largest element** in an unsorted array using the **Quickselect algorithm**.

---

# 2. LRUCache 

Node.js + TypeScript implementation of a **Least Recently Used (LRU) cache** with optional per-entry TTL (Time To Live).

---

## Getting Started

### Clone the repository

```bash
git clone git@github.com:eramosgit/nixplay.git
cd <repo-folder | nixplay>
```

### Install dependencies

```bash
npm install
```

### Run tests

```bash
npm run test
```

### Run demo

```bash
npm run start
```

### Compile TypeScript to JavaScript

```bash
npm run build
```
